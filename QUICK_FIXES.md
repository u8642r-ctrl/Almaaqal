## 🚨 إذا واجهت مشاكل

### المتصفح يعرض نسخة قديمة؟
```bash
# حل 1: امسح Cache المتصفح
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)

# حل 2: افتح وضع التصفح الخاص
Ctrl + Shift + N (Chrome)
Ctrl + Shift + P (Firefox)
```

### لا تزال المواد لا تظهر؟
```bash
# أعد تشغيل السكربت
node scripts/fix-teacher-courses.js

# ثم أعد تحميل الصفحة
```

### أخطاء في البناء؟
```bash
# احذف cache
rm -rf .next

# أعد التثبيت
npm install

# أعد البناء
npm run build

# شغل السيرفر
npm run dev
```
