# تشغيل النظام بدون البيانات الوهمية

## خطوات التشغيل:

### 1. حذف البيانات الوهمية الحالية:
```bash
# قم بتشغيل النص السابق في قاعدة البيانات PostgreSQL
psql -d your_database < delete-dummy-data.sql
```

### 2. إعادة تشغيل التطبيق:
```bash
npm run dev
```

## الحسابات المتاحة:

### حساب الأدمن الوحيد:
- **البريد**: `admin@almaqal.edu.iq`
- **كلمة المرور**: `admin2001`

## كيفية استخدام النظام:

### 1. إضافة المدرسين:
- سجل دخول بحساب الأدمن
- اذهب لصفحة "إدارة الأساتذة"
- أضف المدرسين الجدد

### 2. إضافة الطلاب:
- من نفس لوحة الأدمن
- اذهب لصفحة "إدارة الطلاب"
- أضف الطلاب الجدد

### 3. إضافة المواد:
- من لوحة الأدمن
- اذهب لصفحة "إدارة المواد"
- أضف المواد واربطها بالمدرسين

### 4. للمدرسين:
- سجل دخول بحسابك الخاص
- عند أول دخول، ستُربط المواد المتاحة تلقائياً
- يمكنك إدارة موادك من "موادي الدراسية"

### 5. للطلاب:
- سجل دخول بحسابك
- يمكنك التسجيل في المواد من "السجل الأكاديمي"

## ملاحظات مهمة:
✅ تم إزالة جميع البيانات الوهمية
✅ النظام يعمل بدون بيانات افتراضية
✅ المدرسين الجدد سيحصلون على مواد تلقائياً عند أول دخول
✅ الأدمن يمكنه إدارة كل شيء من لوحة التحكم

## في حالة المشاكل:
- تأكد من تشغيل قاعدة البيانات
- تأكد من صحة متغيرات البيئة (.env)
- استخدم حساب الأدمن لإضافة البيانات الجديدة
